# Day 78 - Question 1 Solution

## 🎯 Problem
Minimum Spanning Tree using Prim’s Algorithm

## 💡 Approach
Use Prim’s algorithm with an adjacency matrix; repeatedly pick the unvisited vertex with the smallest connecting edge, add its weight to the total, and relax edges to remaining vertices.

## ⏱️ Time Complexity
O(n^2)

## 💾 Space Complexity
O(n^2)
