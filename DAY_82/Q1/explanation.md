# Day 82 - Question 1 Solution

## 🎯 Problem
Upper Bound & Lower Bound Implementation

## 💡 Approach
Use binary search twice: once to find the first index where arr[i] >= x (lower bound) and once for the first index where arr[i] > x (upper bound).

## ⏱️ Time Complexity
O(log n)

## 💾 Space Complexity
O(1)
