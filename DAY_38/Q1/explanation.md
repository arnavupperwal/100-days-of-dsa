# Day 38 - Question 1 Solution

## 🎯 Problem
Deque Operations

## 💡 Approach
Use a circular array to implement a deque; head and tail indices wrap around. Each operation updates indices in O(1) and outputs requested values.

## ⏱️ Time Complexity
O(1) per operation

## 💾 Space Complexity
O(N) with N=200
