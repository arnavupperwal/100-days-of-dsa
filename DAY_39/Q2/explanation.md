# Day 39 - Question 2 Solution

## 🎯 Problem
Top K Frequent Elements

## 💡 Approach
Sort the array, count consecutive equal values to get frequencies, sort the (value,frequency) pairs by frequency descending, and return the first k values.

## ⏱️ Time Complexity
O(n log n)

## 💾 Space Complexity
O(n)
