# Day 46 - Question 1 Solution

## 🎯 Problem
Level Order Traversal

## 💡 Approach
Read the number of nodes and the array representation of the tree, then print the array elements in order which is the level‑order traversal for a complete binary tree stored in an array.

## ⏱️ Time Complexity
O(n)

## 💾 Space Complexity
O(n)
