# Day 61 - Question 1 Solution

## 🎯 Problem
Graph Representation Using Adjacency Matrix

## 💡 Approach
Read n and m, initialize a 100x100 matrix to 0, set entries for each edge (both directions for undirected), then print the matrix.

## ⏱️ Time Complexity
O(n^2 + m)

## 💾 Space Complexity
O(n^2)
