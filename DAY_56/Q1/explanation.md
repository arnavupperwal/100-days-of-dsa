# Day 56 - Question 1 Solution

## 🎯 Problem
Symmetric Binary Tree Check

## 💡 Approach
Check symmetry by treating the array as a binary heap; recursively compare left child of one node with right child of the other and vice versa, treating indices beyond n as NULL.

## ⏱️ Time Complexity
O(n)

## 💾 Space Complexity
O(n)
