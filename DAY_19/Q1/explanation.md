# Day 19 - Question 1 Solution

## 🎯 Problem
Pair Sum Closest to Zero

## 💡 Approach
Sort the array and use two pointers to find the pair with sum closest to zero, updating the best pair whenever a smaller absolute sum is found.

## ⏱️ Time Complexity
O(n^2) for sorting + O(n) two‑pointer search → O(n^2)

## 💾 Space Complexity
O(1)
