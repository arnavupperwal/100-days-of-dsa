# Day 84 - Question 1 Solution

## 🎯 Problem
Implement Insertion Sort

## 💡 Approach
The program reads n and the array, then performs insertion sort by moving elements greater than the key to the right and inserting the key in its correct position. Finally it prints the sorted array.

## ⏱️ Time Complexity
O(n^2)

## 💾 Space Complexity
O(1)
