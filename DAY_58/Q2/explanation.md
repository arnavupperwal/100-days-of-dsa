# Day 58 - Question 2 Solution

## 🎯 Problem
Construct Binary Tree from Preorder and Inorder Traversal

## 💡 Approach
Recursively split preorder and inorder arrays around the root found in inorder; build left and right subtrees using subarray sizes.

## ⏱️ Time Complexity
O(n^2)

## 💾 Space Complexity
O(n)
