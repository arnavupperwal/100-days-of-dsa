# Day 72 - Question 1 Solution

## 🎯 Problem
First Repeated Character

## 💡 Approach
Scan the string, keep a 26‑element array of seen flags; on encountering a character already seen, print it and exit, otherwise mark it seen. If loop ends, print -1.

## ⏱️ Time Complexity
O(n)

## 💾 Space Complexity
O(1)
